# HealthPlus
 HealthPlus is a fitness app developed in flutter as a semester project.
 
# Table Of Contents
- [Features](#features)
- [Usage](#usage)

# Features

It includes the following features:
 
Steps Tracker,
Calorie Counter,
Diet Tracker,
BMI Calculator,
Sleep Tracker,
Water Intake Reminder,
Detailed daily/weekly/monthly Report.
